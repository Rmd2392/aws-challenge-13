## Despliegue profesional de web estática

Este sitio forma parte del **Reto 13** y demuestra:

- **Hugo** para generar contenido estático
- **AWS Amplify** para construir y desplegar desde un repositorio Git
- **HTTPS** con certificado gestionado
- **DNS** con subdominio delegado: `aws13.cloudlab.cat` → Route53
- Optimización de **caché** y gestión de errores/redirecciones
